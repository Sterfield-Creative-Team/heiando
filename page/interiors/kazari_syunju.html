Kazari-sara Shunju

Kazari-sara Shunju is...

  • This is an ornament plate with a floral design of "shunju."

The flowers are drawn by using different "maki-e" techniques on each design.

Recommended to place in entrances and drawing rooms, it will brighten up the space with its vivid elegance. Restaurants, hotels, and company offices are also suitable.

Variation Code 01 features "shunju" flowers in gold, along with hand fans.
Variation Code 02 features "shunju" flowers in black.
