Platter, Small Plates in Pair, & Bowl / Golden Haze is...

  • These are traditional dinnerware for serving Japanese cuisine.

Their gorgeous design is inspired by haze in traditional Japanese paintings. It is original to Yamada Heiando, with special techniques inherited to just a few craftsmen.

Variation Code 01 is a large round platter.
Variation Code 02 is in a set of 2 small round plates.
Variation Code 03 is a bowl.
